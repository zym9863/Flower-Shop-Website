# Flower Shop Website

[中文版 (Chinese Version)](./README.md)

This is a beautiful and modern flower shop website project, suitable for displaying and selling flowers.

## Features
- Responsive design for both mobile and desktop
- Showcase flower products and images
- Modern and elegant UI

## Directory Structure
- `index.html`: Main page
- `images/`: Image resource folder

## Usage
1. Clone or download this project locally.
2. Open the `index.html` file in your browser to preview the website.

## Dependencies
This project is a static website and requires no additional dependencies.
