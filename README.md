# 花店网页

[English Version](./README_EN.md)

这是一个美丽、现代化的花店网页项目，适合展示和销售鲜花。

## 功能特色
- 响应式设计，适配手机与桌面端浏览体验
- 展示花卉产品信息与图片
- 现代美观的UI界面

## 目录结构
- `index.html`：主页面
- `images/`：图片资源文件夹

## 使用方法
1. 克隆或下载本项目到本地。
2. 使用浏览器打开 `index.html` 文件即可预览网页效果。

## 依赖
本项目为静态网页，无需额外依赖。
